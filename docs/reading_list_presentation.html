Cross-Cutting Insights

Patterns that emerged across multiple categories — the meta-lessons for building the alpha signal agent.

1. Infrastructure Over Intelligence
Multiple sources converge: the orchestration harness, data architecture, and evaluation infra matter more than model capability. Invest in backtesting pipelines and experiment registries, not chasing frontier models.
2. Structural Enforcement > Behavioral Prompting
Constraints that must never be violated (point-in-time integrity, entitlement scoping, compliance) should be enforced by platform infrastructure, not prompts. A single lookahead leak can invalidate an entire research campaign.
3. Evaluation-Gated Self-Improvement
Self-improvement must be governed by held-out validation, not training metrics. No strategy promotion without OOS checks, CSCV/PBO analysis, and human approval.
4. Persistent Knowledge Accumulation
Agents become valuable when they accumulate and structure domain knowledge across sessions — capturing failed hypotheses, feature patterns, regime-specific model choices as searchable institutional memory.
5. Proposal-Critic-Verifier Pattern
Generating many candidates cheaply and selecting rigorously through verification is more robust than generating fewer, higher-quality candidates without empirical validation.
6. Hybrid Deterministic-Agentic Orchestration
Interleave deterministic steps (data retrieval, point-in-time alignment, cost calculation) with agentic steps (hypothesis generation, feature engineering). Pure autonomy and pure workflows both fail.
7. Context Architecture Is the Binding Constraint
Managing information across extended multi-step workflows is the hardest unsolved engineering challenge. For financial agents handling large datasets and regulatory context, this defines the capability ceiling.
8. Agent Reward Hacking = Backtest Overfitting
METR's 16% eval cheating rate maps directly to backtest overfitting. Agents will game metrics if given opportunity — structurally identical to the overfitting problem that has plagued quant finance for decades.
